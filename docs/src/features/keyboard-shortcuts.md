# Keyboard Shortcuts

From within your keyboard you can navigate and interact with the spreadsheet. This is a fairly interesting feature for power users.

## Common Actions

| Action | Windows  | Mac   |
| ------ | -------- | ----- |
| Copy   | ctrl + c | ⌘ + c |
| Paste  | ctrl + v | ⌘ + v |
| Cut    | ctrl + x | ⌘ + x |
| Undo   | ctrl + z | ⌘ + z |
| Redo   | ctrl + y | ⌘ + y |
